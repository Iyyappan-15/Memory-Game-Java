# Memory Game in Java

This is a simple console-based memory game written in Java.
The player flips two cards at a time and tries to match pairs.

## Features
- Console-based game
- Random card shuffle
- Simple user input
- Beginner-friendly code

## How to Run
1. Compile the program:
   javac Hello.java
2. Run the program:
   java Hello

## Technologies Used
- Java
- ArrayList
- Scanner
